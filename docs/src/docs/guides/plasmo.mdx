# Usage with plasmo

## Generate new web extension

Init new extension:

```bash
yarn create plasmo
```

<PackagesInstallation />

Change `popup.tsx` file:

```tsx
import { MantineProvider, Text } from '@mantine/core';

export default function Popup() {
  return (
    <MantineProvider withGlobalStyles withNormalizeCSS>
      <Text>Welcome to Mantine!</Text>
    </MantineProvider>
  );
}
```

All set! Start development server:

```bash
yarn dev
```

Then load the extension into your browser and open the popup.

## Content Scripts UI with custom Emotion Cache

Create `content.tsx` file:

```tsx
import { MantineProvider, Text, createEmotionCache } from '@mantine/core';

import React from 'react';

export const config: PlasmoContentScript = {
  matches: ['https://mantine.dev/*'],
};

const styleElement = document.createElement('style');

const styleCache = createEmotionCache({

      <Text>Welcome to Mantine!</Text>

```

Load the extension into your browser, and navigate to [https://mantine.dev/](https://mantine.dev/). The CSUI should be injected into the page.

For more info about custom emotion cache, see [custom emotion cache guide](/theming/emotion-cache/#server-side-rendering-with-custom-cache). This technique is required to mount the style into the CSUI ShadowDOM.
